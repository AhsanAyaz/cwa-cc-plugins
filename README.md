# code-with-ahsan

Claude Code plugin marketplace by [AhsanAyaz](https://github.com/AhsanAyaz).

## Add this marketplace

```
/plugin marketplace add AhsanAyaz/code-with-ahsan
```

## Plugins

| Plugin | Description |
|--------|-------------|
| [claude-halal-code](https://github.com/AhsanAyaz/claude-halal-code) | Displays a Quranic ayah at the start of every Claude Code session |

## Install a plugin

```
/plugin install claude-halal-code@code-with-ahsan
```
